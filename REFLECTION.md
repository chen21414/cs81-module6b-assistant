What made your assistant “personal” to you?

I used tasks that are connected to my actual life. These tasks included completing my JavaScript assignment, reviewing codes, and studying for the exam. I also named the assistant MikeBot to connect it to my own name. 

What challenges did you face using this?

One challenge was understanding exactly what this referred to inside each method. For example, when myAssistant.addTask() is called, this.tasks refers to the tasks array inside myAssistant. It was initially confusing because the meaning of this depends on how the function is called, but using it in couple methods helped me understand the idea behind it.

Which method would you improve or expand?

I would expand the completeTask() method. Currently, it always completes the first task in the array. I would improve it by allowing the user to select a specific task.
